Discuss the scope of var, let, and const

var declaration global scoped or function scoped.var variables can be updated and re-declared.
let and const are block scoped.let variables can be updated but not re-declared.
const variables can not be updated not re-declared

Tell us the use cases of null and undefined

Null is an assignment value.When the value of a variable is empty then the value of that variable is null.
Undefined means a variable has been declared but has yet not been assigned a value.
